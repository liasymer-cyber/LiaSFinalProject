# MeToo, Rape Reporting, and Media Sentiment

**QSS 20 Final Project — Dartmouth College, 2026**  
**Author:** Lia Symer
**Website Link:** https://reporting-metoo-mediasentiment.vercel.app/ 

A quantitative analysis of whether the #MeToo movement (October 2017) produced measurable changes in official rape reporting rates and news headline sentiment across the United States.

---

## Research Questions

1. Did reported rape rates across U.S. states change or rise more than other violent crimes between 2014 and 2019, spanning the #MeToo inflection point?
2. Did the sentiment of news headlines about sexual assault shift after October 2017, toward less negative or more survivor-centered framing?

---

## Key Findings

| Finding | Result |
|---------|--------|
| Rape reporting rate, 2014→2019 | +16.4% (36.6 → 42.6 per 100k), p = 0.018 |
| DiD: rape vs. other violent crimes (post × is_rape) | +5.71 per 100k, p < 0.001 |
| DiD: rape vs. murder only (robustness check) | +4.15 per 100k, p < 0.001 |
| Headline sentiment shift (VADER) | +0.10, p < 0.0001, r = 0.24 |
| Headline sentiment shift (LLM) | +0.50, p < 0.0001, r = 0.14 |

## Notebooks
### [`01-LoadingDataRapeStats_merged.ipynb`](code/01-LoadingDataRapeStats_merged.ipynb)
Loads and cleans the FBI UCR `.xls` files for 2014–2019. Strips footnote suffixes from state names, handles the 2013 rape definition change, extracts rape rate per 100,000 for all 50 states, and produces the choropleth maps comparing 2014 and 2019 rates.
### [`02-WordAnalysisMeToo.ipynb`](code/02-WordAnalysisMeToo.ipynb)
Loads and filters the four annual MediaCloud CSVs. Applies 14 keyword regex patterns across headline and URL fields to isolate sexual assault–relevant coverage (115,525 matches from 470,225 total). Performs VADER sentiment analysis on ~68,000 headlines with a domain-customized lexicon (e.g. *survivor: +1.0*, *alleged: −1.0*). Produces the VADER sentiment time series figure.
### [`03-SentimentwithOllama.ipynb`](code/03-SentimentwithOllama.ipynb)
Draws a stratified random sample of 2,500 pre-MeToo and 2,500 post-MeToo headlines and scores each using `llama3.2` via the local Ollama API. Each headline is assigned a float from −1 (victim-blaming/negative) to +1 (survivor-centered/empowering). Runs the Mann-Whitney U test comparing pre/post distributions for both VADER and LLM scores.
### [`04-DiDMeToo.ipynb`](code/04-DiDMeToo.ipynb)
Difference-in-differences analysis using other violent crimes (murder, robbery, aggravated assault) as the control outcome. Handles the FBI's 2017 column shift (removal of the legacy rape definition) by detecting column positions dynamically per file. Constructs a balanced long panel of 1,200 state × crime-type × year observations, fits two-way fixed effects models with HC3 robust standard errors, and produces the event study plot.
